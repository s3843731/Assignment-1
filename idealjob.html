Ideal Job


Penetration Tester at Deloitte


Position

This position will include tasks that involve recognising any flaws in the security within the
IT environment of clients, highlighting any possible exposures and generating strategies for rectification. Another aspect of
this position is acting as a support and mentor to the other team members within the workplace. Some of the security aspects
include the following; networking and host layer vulnerability assessments, firewall, networking and security device reviews,
web application assessments, Application Programming Interface (API) assessments and social engineering. A reason this job
is appealing to me is that the job is located in Melbourne, which is where I am currently situated, so location is not a
problem. Also, Deloitte is an accounting organisation which is were I could apply both my economic and finance skills and
also my IT skills, which I will gain through my tertiary studies. The position is also entry level with a full time contract,
which promises a sense of job security to an extent.

Skills, Qualifications and Experience


To be qualified for this position, the major criteria include the following:
- having a CREST Certified Tester (CCT) in either Infrastructure or Web Applications or similar or be in a position and level to attain certification - experience of working with applications that perform a range of business functions,
- capability of understanding and evaluating applications from a technical and business perspective,
- experience in conducting web application penetration testing and conducting penetration,
- testing with a novel and inquisitive approach.


Deloitte is looking for an application who has experience in one or more of the following areas:
- Reverse engineering
- Web applications
- API's and Microservices
- Exploit Development
- Application vulnerability assessment
- Mainframe systems
- Mobile platforms (iOS/Android/Windows/etc)
- Social engineering
- Endpoint protection
- Practical experience to security appliances such as firewalls, proxies, NIPS/HIPS and network security applications
- Knowledge of web concepts such as Ajax, XML, SOAP, WS-Security
- Familiarity with Open Source Security Testing Methodology Manual (OSSTMM), Open Web Application Security Project (OWASP) and National Institute of Standards and Technology (NIST) Special Publications
- Familiarity with penetration testing and vulnerability tools such as backtrack, dsniff nessus, nmap, MetaSploit, Hailstorm, WebInspect etc.
- Strong programming experience with Visual Basic and C/C++ or Java languages
- Networking: LAN, WAN, interworking technologies
- Good understanding of IaaS environments like Azure, AWS and GCP
