My IT Project

Overview

My IT project will comprise of developing a smartphone application that can be used by people for organisation and health. This application is aimed at students and others who have difficulty managing their time and activities. The application will be developed by the use of the programming language Java, C++ or of the sort. The application will have multiple aspects, including; a timer feature, studying tips and exercise tips, and physical exercises. These can be used to help the productivity of the user, whether it is for studying or better organisation and time management.


Motivation

This IT project will be useful as people can use it to increase productivity, organisation and time-management which can lead to better motivation. As a current student, I have found it very difficult to find motivation in high school and university. Also, it was quite difficult to manage school work with my job and other activities. The motivation behind this app is so it can help others be more productive and healthy going through educational studies as it can be a stressful time for most people, if not all. Furthermore, this app can help in motivating people stay healthy mentally and physically.

Tools and Technologies

This application will require application development software such as AppyPie and AppSheet, software that supports different platforms like, Apple, Android, Mac, Windows and Linux. The programming language that can be used to develop this application is C++ or Java as these two programming languages are compatible with multiple platforms. To make the application more user-friendly and more appealing, front-end development tools will need to implemented to maximise user experience. A media player will need to be implemented for playing audio files. Hardware that may prove useful to this application is a Bluetooth radio to allow for Bluetooth connections for audio.
